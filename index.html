Windows 10 : c'est officiel!
  • Le grand retour du menu démarrer
  • Le gestionnaire des tâches remis à neuf
  • Les applications enfin en mode fenêtré

Annoncé durant le mois de Septembre 2014, la version finale de Windows 10 ne sera pas dévoilé avant la conférence du Build 2015, soit 2 ans après le lancement de Windows 8.1 lors du même évenement. Pour ce nouvel opus du célèbre système d'exploitation, le nouveau né du géeant Microsoft décide de consolider ses points forts ; Windows 10 se trouve grandement optimisé pour les ordinateurs ( comparativement a la version 8.1 et ses tuiles ) et se veut plus facile d'utilisation que jamais !
